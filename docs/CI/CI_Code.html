Class CI_Code

Class Overview
Class Overview | Descendants | Inherited Properties, Constants, and Methods | Property Summary | Properties Detail | Method Summary | Methods Detail

Абстрактный класс для обработки кода. Содержит общую логику рпоцесса оптимизации для всех типов кода.

Расширяет класс кэша используя его функционал для хранения различных кусков кода.

Является абстрактной фабрикой.

optimizeCode  [line 51]

  void optimizeCode( )

Реализует общую лугику обработки любого кода системой.

Определяет размер кода до обработки, записывает его в $sizeBefore. Если код ранее уже был обработан, то возвращает результаты из кеша, если нет, то запускует процесс оптимизации.

После получения результата оптимизации записывает код его в $optimizedCode, определяет размер после обработки и записывает его в s$sizeAfter.

Делает пометку в журнал файле о статистике оптимизации.
